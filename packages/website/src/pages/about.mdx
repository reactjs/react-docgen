# About react-docgen

Nextra was initially created by [Facbook](https://www.facebook.com) member
[Felix Kling](https://felix-kling.de) in 2015. Since 2016,
[Daniel Tschinder](https://danieltschinder.com) contributed to the project and
in 2019 took over as a core maintainer.

The project was initially licensed with Facebook's version of the BSD license.
With version 5.0 in 2019 the license was changed to MIT.

## Team

Currently, the project is maintained by
[Daniel Tschinder](https://danieltschinder.com). You can check out the full list
of contributors on
[GitHub](https://github.com/reactjs/react-docgen/graphs/contributors).
